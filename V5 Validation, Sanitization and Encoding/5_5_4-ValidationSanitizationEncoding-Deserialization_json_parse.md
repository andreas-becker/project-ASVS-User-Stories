# V5.5 Deserialization Prevention

## ASVS: 5.5.4

## ASVS Requirement description

Verify that when parsing JSON in browsers or JavaScript-based backends,
JSON.parse is used to parse the JSON document. Do not use eval() to parse
JSON.
