# V1.7 Errors, Logging and Auditing Architecture

## ASVS: 1.7.1

## ASVS Requirement description

Verify that a common logging format and approach is used across
the system.
([C9](https://owasp.org/www-project-proactive-controls/#div-numbering))

## User Story

**Feature_Name**: Common Logging Format Implementation

**Story**:
As a Security Engineer\
I want to ensure that a common logging format and approach is used across the system\
So that we can maintain consistency, readability, and ease of analysis in our logs

## Scenario

**Scenario_name**: Implementing a common logging format across the system

**Gherkin syntax**:

```gherkin
Given our system's logging processes
When I implement a common logging format and approach
Then consistency, readability, and ease of analysis in our logs are maintained
```
